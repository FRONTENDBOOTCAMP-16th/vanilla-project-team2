React에서 useEffect와 useLayoutEffect의 차이점이 궁금합니다

1시간 전

초보개발자

7

React를 공부하다가 useEffect와 useLayoutEffect의 차이점에 대해 궁금한 점이 생겼습니다.
공식 문서를 읽어봤는데, 실행 시점이 다르다는 것은 알겠지만 실제로 어떤 상황에서 각각을 사용해야 하는지 명확하게 이해가 되지 않습니다.
실무에서 사용하시는 분들의 경험담과 구체적인 예시를 들어 설명해주시면 정말 감사하겠습니다! 

              // useEffect 예시
  useEffect(() =< {
    // DOM 업데이트 후 실행
    console.log('useEffect 실행');
  }, []);
  
  // useLayoutEffect 예시
  useLayoutEffect(() => {
    // DOM 업데이트 후 즉시 동기적으로 실행
    console.log('useLayoutEffect 실행');
  }, []);
위 코드처럼 사용하면 되는 건 알겠는데, 구체적으로 언제 useLayoutEffect를 써야 하는지 실전 예시가 궁금합니다.

코멘트

  1. 30분 전 답변자1

    좋은 질문이네요! useEffect는 DOM 업데이트 후 비동기적으로 실행되고, useLayoutEffect는 DOM 업데이트 후 동기적으로 실행됩니다.

  2. 1시간 전 답변자2

    useLayoutEffect는 DOM 변경을 읽고 동기적으로 다시 렌더링해야 할 때 사용하면 좋습니다. 예를 들어 스크롤 위치를 측정하거나 DOM 노드의 크기를 읽어야 할 때 유용합니다.

  3. 2시간 전 답변자3

    대부분의 경우 useEffect를 사용하면 되고, useLayoutEffect는 화면 깜빡임을 방지해야 하는 특수한 경우에만 사용하는 것이 좋습니다.
